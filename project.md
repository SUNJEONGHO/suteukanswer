# 수능특강 문제 풀이 해설

이 프로젝트는 수능특강 문제를 풀이하는 웹 서비스야.

## 사용해야하는 것
- next.js
- mongodb (mongodb mcp 사용)
- tailwind css


## 넣어야 하는거
1. 문제 검색(단원(String), 문제번호(number) )를 선택해서 문제를 검색할 수 있도록
2. 해당 문제를 선택하면 관리자가 미리 업로드해둔 html 파일을 띄어줄것
3. html 코드를 입력할 수 있는 관리자 페이지를 만들어 줄것

## 플랜짜기 가이드
순서대로 고민해서 플랜을 차근차근 자세히 작성해주면 됨
1. 자세한 디테일 질문(한국어로 질문)
2. 데이터베이스 스키마(로그인, 회원가입 기능 필요 없음)
3. 페이지 구성
4. 풀이 업로드 페이지
